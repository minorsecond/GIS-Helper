##Spatial Tools  [![Build Status](http://ci.imspatial.me/buildStatus/icon?job=GIS Helper)](http://ci.imspatial.me/job/GIS%20Helper/)

A tool to facilitate with GIS tasks.

### How do I get set up? ###

* Requires Python 3.5.
* Clone repository and run 'pip3 install -r requirements.txt'.
* In Windows, run the build.bat file. Pyinstaller must be present.
